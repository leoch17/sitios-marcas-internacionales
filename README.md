# Sitios de Marcas Internacionales 🌍

Este repositorio principal actúa como un contenedor organizado para diversos proyectos de desarrollo web enfocados en marcas internacionales. En lugar de copiar el código, cada proyecto se gestiona como un **Git Submodule** independiente, manteniendo su propio historial de versiones y despliegue.

## 🚀 Proyectos Incluidos

Haz clic en los enlaces a continuación para ir directamente al repositorio específico de cada marca:

- [**Dunlop**](https://github.com/leoch17/dunlop/tree/10b9642ad903fbbf380b945cffc3a553fc4fc79e) - Proyecto de migración a Next.js y optimización de componentes.
- [**Yokohama**](https://github.com/leoch17/yokohama-landing-nextjs/tree/51e53690e9503d56c9a84f818fed3a13b8f5ebe8) - Landing page profesional con diseño moderno y responsivo.
- [**Gulf**](https://github.com/leoch17/gulf/tree/c1a7b0c8d806376313ba168bcbe13fb2f4823091) - Desarrollo web y presencia digital para la marca Gulf.
- [**Prinx**](./prinx) - Desarrollo de sitio web para la marca de neumáticos Prinx.

---

## 🛠️ Notas para Desarrolladores

Si acabas de clonar este repositorio por primera vez, recuerda que las carpetas de los submódulos aparecerán vacías inicialmente. Para descargar el contenido de todos los proyectos, utiliza:

```bash
git clone --recursive [https://github.com/leoch17/sitios-marcas-internacionales.git](https://github.com/leoch17/sitios-marcas-internacionales.git)
```
